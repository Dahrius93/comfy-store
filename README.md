# Comfy Store

E-commerce di arredamento costruito con React. Permette di navigare i prodotti, aggiungerli al carrello, effettuare il login e completare ordini.

**Demo:** [my-new-comfy-store.netlify.app](https://my-new-comfy-store.netlify.app/)

## Stack

- React + React Router
- Redux Toolkit
- React Query
- Tailwind CSS + DaisyUI

## Funzionalità

- Catalogo prodotti con filtri e paginazione
- Carrello persistente
- Autenticazione (login/registrazione)
- Checkout e storico ordini
